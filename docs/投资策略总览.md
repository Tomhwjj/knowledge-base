## 二、交易模式与战法

### 1. 绿盘低吸模式（曹小强最常用）
- 选股苛刻：必须有强有力的上涨逻辑 → 能控制回撤 → 能安全退出
- **适合震荡市/弱市**，不适合牛市主升浪

### 2. 打板 / 首板
- 熊市一进二：好票买不到，买到的大概率没溢价，还可能吃大面
- 首板窗口只有 ~7 秒，过了就没溢价或炸板

### 3. 潜伏等轮动
- 熊市没有主线、没有持续性，板块快速轮动
- 选低位板块潜伏，等轮动反弹，**三五个点就走**

### 4. 龙头战法
- 牛市好用，短期内赚过 150%；**熊市/地狱模式同样战法短期亏 50%+**
- 关键是判断市场处于什么阶段

### 5. 趋势 / 右侧交易 vs 接飞刀
- 下跌中继 ≠ 抄底，等信号确认再进

## 三、选股条件（三层过滤）

```
第一层：上涨逻辑 — 为什么涨？（题材、业绩、板块联动）
第二层：回撤控制 — 万一跌了，能亏多少？有安全垫吗？
第三层：安全退出 — 有流动性吗？能顺利出货吗？
```

具体维度：
- 板块整体趋势是否走好（龙头 + 市值中军是否配合）
- 涨停板数量多但无市值中军大涨 → 板块可能衰竭（"拍毕业照"）
- 个股走势是否明显强于板块（"该跌不跌就是强"）
- 熊市资金对业绩要求高，年报季要回避"杀白马"

## 四、止盈止损纪律

| 规则 | 说明 |
|------|------|
| **止盈** | 赚自己认知内的钱，"三五个点就走"不丢人。卖飞了也别后悔——这次运气赚的，下次可能连本带利还回去 |
| **止损** | 大亏的根源就是**不止损**。预设止损价，到了就执行，不拖泥带水 |
| **不补仓** | 走势不符合预期 → 止损走人，不是盲目补仓摊低成本，更不是装死不看 |

## 五、市场环境判断（四大模式）

| 模式 | 说明 | 策略 |
|------|------|------|
| **简单模式**（牛市） | 普涨、持续性好 | 龙头战法、追涨都好用 |
| **普通模式**（震荡偏强） | 有主线、有板块效应 | 低吸 + 趋势跟踪 |
| **困难模式**（震荡偏弱） | 板块快速轮动、无持续性 | 潜伏等轮动，小盈即走 |
| **地狱模式**（熊市） | 每天跌多涨少 | 轻仓/空仓、防守为主 |

如何判断企稳：龙头股止跌 → 前期主线板块止跌 → 有新的持续性板块站出来 → 北证50等先行指标走好。

## 六、交易心态与习惯

1. **记录交易日志**：记录每次交易的心态、决策过程，复盘自己的弱点
2. **不抄作业**：别人的模式不一定适合自己，找到自己最舒服的模式持续迭代
3. **主力专搞散户心态**：让你在错误时间、错误心态下做错误决定
4. **做简单而正确的事**：严格执行纪律，不心存侥幸
5. **流水不争先，争的是滔滔不绝**：稳定复利 > 短期暴利
