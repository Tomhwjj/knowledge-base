### 3. trading-system：选股+交易日志 Phase 1 ✅
**架构**：`stock-workflow` 升级为 `trading-system`

```
Claude Code 对话 + Web 看板 → FastAPI 后端 → SQLite + 东方财富 API
```

**新增模块**：
- `journal/models.py` — trades 表（7手动字段 + 自动补全 + 盈亏计算）
- `journal/auto_fill.py` — 东方财富自动补全（名称/量/板块/大盘环境）
- `journal/routes.py` — FastAPI 8 接口
- `main.py` — 一键启动 `http://localhost:8765`
- `web/` — 暗色主题看板（交易列表 + 录入表单 + Chart.js 盈亏曲线）
- `skill/TRADING.md` — 选股+交易合并 Skill

**对话入口**：
- "日报" — 收盘速报
- "记一笔 买入 600900 26.5 14:30 半仓 波段" — 录入交易
- "本月统计" / "当前持仓"

**独立仓库**：https://github.com/Tomhwjj/trading-system

## 关键技术决策
| 决策 | 选择 | 原因 |
|------|------|------|
| 数据源 | 东方财富免费API | 零成本，够用 |
| 数据库 | SQLite | 个人工具，单文件最省心 |
| 前端 | 单页HTML + Chart.js | 极简，无框架依赖 |
| 架构 | 三层（对话/Web/后端） | 互补，不冲突 |
