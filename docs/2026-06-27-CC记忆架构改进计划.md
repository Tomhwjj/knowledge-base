## P0: 今天（4项）

### 1. 水印估算精度（缺陷 #4）

**当前**: `字符 × 0.35` 一元估算，中英混排误差 >30%
**改进**: 区分中英文分别估算

```python
def count_tokens_fallback(text):
    cn = len(re.findall(r'[一-鿿　-〿＀-￯]', text))
    en_chars = len(re.findall(r'[a-zA-Z0-9]', text))
    other = len(text) - cn - en_chars
    return int(cn * 1.8 + en_chars * 0.25 + other * 0.5)
```

**文件**: `waterline.py` 修改 `count_tokens` 的回退分支

### 2. 健康监控告警阈值（缺陷 #6）

### 5. 图谱共现词扩展（缺陷 #1）

**改进**: 对同一文档中高频共现的技术词，自动建立隐式边

```python
# graph.py 新增
def build_cooccurrence(docs, top_n=50):
    """扫描所有文档，提取 top_n 高频词，建立共现矩阵"""
    pairs = []
    for doc in docs:
        words = set(jieba.analyse.extract_tags(doc, topK=10))
        for w1 in words:
            for w2 in words:
                if w1 < w2:
                    pairs.append((w1, w2))
    # 统计 pair 频次，高频 pair 视为隐式边
    ...
```

**文件**: `graph.py` 新增函数，graph_cache.json 加 `cooccurrence` 字段

### 6. 写入前冲突检测（缺陷 #5）

**当前**: memory_digest.py --check 只检查相似度（去重），不检查矛盾
**改进**: `--check` 同时输出语义相似度和潜在冲突

```python
def check_conflict(summary, existing_summaries):
    """相似度低但主题高度重合 → 潜在冲突"""
    # 主题词提取
    new_topics = set(jieba.analyse.extract_tags(summary))
    for old_sum, old_file in existing_summaries:
        old_topics = set(jieba.analyse.extract_tags(old_sum))
        overlap = new_topics & old_topics
        sim = cosine_sim(embed(summary), embed(old_sum))
        # 主题重叠 > 50% 但相似度 < 0.5 → 可能是冲突
        if len(overlap) / min(len(new_topics), len(old_topics)) > 0.5 and sim < 0.5:
            return {"conflict": True, "file": old_file, "overlap_topics": list(overlap)}
    return None
```

**文件**: `memory_digest.py` 扩展 `check_dedup`

### 7. 分标签类型检索权重（缺陷 #2）

**当前**: RRF k=60 统一对待三路
**改进**: query.py 支持 `--bias` 参数

| 场景 | BM25权重 | 向量权重 | 图谱权重 |
|------|:--:|:--:|:--:|
| 默认 | 1.0 | 1.0 | 1.0 |
| --bias rule | 1.5 | 0.8 | 0.7 |
| --bias issue | 0.7 | 1.5 | 0.8 |

**文件**: `query.py` RRF函数加权重参数

### 8. 启动延迟加载（改进 #2）

**当前**: 启动即加载所有 #hot
**改进**: CLAUDE.md 改为

```
启动时只跑 waterline.py。
等用户发第一条消息后，根据消息主题决定:
  - 技术/编码类 → 偏 #mem-rule + #mem-issue
  - 架构/决策类 → 偏 #mem-decision
  - 通用/闲聊 → 只加载 #mem-rule（约束）
再跑 memory_load.py --full 或 query.py。
```

### 9. 连续时间衰减（改进 #5）

**当前**: hot/warm/cold 三档阶梯
**改进**: Reranker 后乘 `e^(-λt)`，半衰期30天

```python
import math
decay = math.exp(-0.023 * age_days)  # λ = ln(2)/30 ≈ 0.023
adjusted_score = rerank_score * decay
```

**文件**: `query.py` retrieve() 的 rerank 后加衰减因子
